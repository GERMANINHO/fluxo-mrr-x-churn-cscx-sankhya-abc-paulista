Fluxo MRR x Churn — Setup + Integração

Este documento descreve como padronizar o fluxo no GitHub Pages, manter a planilha como sub-link dentro do fluxo e apontar o Hub para o site do fluxo (e não direto para o Sheets).

Abrir planilha — MRR x Churn Abrir site do Fluxo

Atalhos

0) Visão geral 1) Criar repositório 2) Ativar GitHub Pages 3) Criar o index.html 4) Conteúdo do fluxo 5) Atualizar o Hub 6) Checklist final

0) Visão geral (padrão correto)

A planilha (Google Sheets) deve ser um sub-link dentro do seu fluxo (página do Git). O Hub deve apontar para a URL do GitHub Pages do Fluxo.

✅ Resultado esperado: Hub → Fluxo (GitHub Pages) → Planilha

1) Criar um novo repositório no GitHub

  • GitHub → New repository
  • Nome no padrão: fluxo-mrr-x-churn-cscx-sankhya-abc-paulista
  • Marcar: Public e Add a README
  • Criar repositório

2) Ativar o GitHub Pages

  • Repositório → Settings → Pages
  • Build and deployment:
    • Source: Deploy from a branch
    • Branch: main
    • Folder: /(root)
  • Salvar e aguardar a URL do Pages aparecer

URL padrão do fluxo: https://germaninho.github.io/fluxo-mrr-x-churn-cscx-sankhya-abc-paulista/ OK

3) Criar o arquivo do Fluxo (index.html)

Opção A (mais rápida): pelo próprio GitHub

  • Repo → Add file → Create new file
  • Nome: index.html
  • Colar o conteúdo do fluxo
  • Commit

Opção B (via Git no PC):

git clone https://github.com/germaninho/fluxo-mrr-x-churn-cscx-sankhya-abc-paulista.git
cd fluxo-mrr-x-churn-cscx-sankhya-abc-paulista
git add .
git commit -m "Cria Fluxo MRR x Churn"
git push

4) Conteúdo do Fluxo (processo + link do Excel)

Dentro do index.html do fluxo, manter:

  • Título: Fluxo MRR x Churn
  • Objetivo do fluxo
  • Pré-requisitos (Unidade, período, etc.)
  • Passo a passo do processo (itens 1 a 6)
  • Link do Google Sheets (como botão)
  • (Opcional) “Validações” e “Erros comuns” (competência/classificação/divergência)

  Abrir planilha — MRR x Churn
</a>

5) Atualizar o Hub (apontar para o Fluxo, e não direto pra planilha)

No Hub, o item do menu deve apontar para o GitHub Pages do Fluxo. A planilha fica dentro do fluxo como sub-link.

Snippet para colar no Hub:

<!-- ✅ NOVA ABA: Fluxo MRR x Churn -->
<li>
  <a class="nav-link"
     href="https://germaninho.github.io/fluxo-mrr-x-churn-cscx-sankhya-abc-paulista/">
    <i class="ion-arrow-graph-up-right"></i>
    <span class="nav-text">Fluxo MRR x Churn</span>
  </a>
</li>

6) Checklist final (100% correto)

  • OK Repositório criado com nome padrão
  • OK Pages habilitado (main / root)
  • OK index.html publicado
  • Fluxo contém:
    • Passo a passo (1 a 6)
    • Link do Sheets
    • Regras de conferência (Dez–Nov etc.)
  • Hub aponta para o GitHub Pages do Fluxo
  • Teste: Hub → Fluxo → Planilha
